# Руководство по новому дизайну CryptoTax Wallet

## Обзор

Создан совершенно новый премиум минималистичный дизайн для криптокошелька с разделением на страницы. Дизайн вдохновлен современными криптокошельками (Trust Wallet, MetaMask) и использует темную цветовую схему с акцентами.

## Структура

### Файлы

- **HTML**: `wallet_nalog/templates/wallet_nalog/index.html` - главный шаблон SPA
- **CSS**: `wallet_nalog/static/wallet/css/main.css` - все стили
- **JavaScript**:
  - `wallet_nalog/static/wallet/js/router.js` - роутинг между страницами
  - `wallet_nalog/static/wallet/js/api.js` - API клиент с автообновлением токенов
  - `wallet_nalog/static/wallet/js/app.js` - основная логика приложения

### Страницы

1. **Авторизация/Регистрация** (`/`)
   - Минималистичная форма с переключением между входом и регистрацией
   - Премиум дизайн с градиентами

2. **Главная (Dashboard)** (`/#dashboard`)
   - Карточки со статистикой (баланс, транзакции, налоги)
   - Информация о кошельке
   - Быстрые действия

3. **Транзакции** (`/#transactions`)
   - Список всех транзакций
   - Визуальное различие входящих/исходящих
   - Автоматическое обновление

4. **Налоги** (`/#tax`)
   - Расчет налога за месяц
   - Помесячный отчет
   - Итоговый налог

## Цветовая схема

- **Фон**: Темный (#0A0E27)
- **Поверхности**: Темно-синие оттенки (#13182E, #1A1F3A)
- **Акцент**: Синий (#0088CC)
- **Успех**: Зеленый (#10B981)
- **Ошибка**: Красный (#EF4444)

## Особенности дизайна

1. **Минимализм**: Чистый интерфейс без лишних элементов
2. **Премиум ощущение**: Плавные анимации, тени, градиенты
3. **Адаптивность**: Полностью адаптивный дизайн для мобильных устройств
4. **Темная тема**: Современная темная тема для комфортной работы

## Навигация

- Боковая панель (Sidebar) с основными разделами
- Верхняя панель (Topbar) с информацией о статусе кошелька
- Hash-based роутинг для SPA

## API Интеграция

Все API вызовы обрабатываются через `api.js`:
- Автоматическое обновление токенов при истечении
- Обработка ошибок
- Единый интерфейс для всех запросов

## Использование

1. Запустите Django сервер: `python manage.py runserver`
2. Откройте в браузере: `http://localhost:8000/` или `http://localhost:8000/app/`
3. Старая версия доступна по: `http://localhost:8000/test/`

## Технологии

- Чистый JavaScript (без фреймворков)
- CSS3 с CSS переменными
- TON Connect для подключения кошельков
- Django Templates для базовой структуры
