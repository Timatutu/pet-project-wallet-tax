    """–ü–æ–ª—É—á–µ–Ω–∏–µ –∏—Å—Ç–æ—Ä–∏–∏ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–π –∞–∫–∫–∞—É–Ω—Ç–∞"""

    current_hash = account_state.last_transaction_hash
    
    while len(transactions) < limit and current_lt:

                address=address,
                lt=current_lt,
                hash=current_hash,
                limit=10

            if txs:

                current_hash = txs[-1].prev_trans_hash

            print(f"–û—à–∏–±–∫–∞ –ø—Ä–∏ –ø–æ–ª—É—á–µ–Ω–∏–∏ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏–π: {e}")

import asyncio

            lambda: client.get_masterchain_info(),
            max_retries=3
        )
